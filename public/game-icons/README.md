# 遊戲圖標目錄

請將以下遊戲的 logo 圖片放在此目錄：

1. **csgo.png** - CS:GO 遊戲圖標
2. **apex.png** - Apex Legends 遊戲圖標
3. **lol.png** - League of Legends (英雄聯盟) 遊戲圖標
4. **pubg.png** - PUBG 遊戲圖標
5. **valorant.png** - VALORANT (特戰英豪) 遊戲圖標

## 圖片要求

- 格式：PNG（支援透明背景）
- 建議尺寸：512x512 或更高（會用 CSS 縮放）
- 檔案命名：使用小寫，如 `csgo.png`、`apex.png` 等

## 使用方式

圖片會自動通過 `components/GameIcon.tsx` 組件載入，並根據上下文自動調整大小。
